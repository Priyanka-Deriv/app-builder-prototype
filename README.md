# App Builder — Reference Prototype

Interactive HTML prototype for the Deriv API Dashboard App Builder (Phase 1: Digits).

## What this covers

- **Step 1 — Configure:** App ID, Auth Redirect URL, Affiliate Token, Markup % (0–5%)
- **Step 2 — Customise:** Logo upload, brand colour picker with live preview
- **Step 3 — Host & Export:** Netlify/Vercel deploy, GitHub push, ZIP download

## View it

Open `index.html` in any browser, or visit the GitHub Pages URL after deploying.
